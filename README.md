# Installation
1. `git clone https://github.com/davidmurray/CIV6707A-TP1.git` (ou avec un client git, peu importe...)
2. Installer `yarn`.
3. Executer `yarn` pour installer les librairies nécessaires.
4. Valider les tests unitaires: `yarn test`.
